SELECT PROJECTS

More on projects

Offline Video Facial Recognition/Object Detection

Comapared with its camera feed application counterpart, this application opens doors to more intensive offline parallel processing with supercomputing clusters, leveraging heavier neural networks, and ensures precise frame-by-frame operations. Tens of objects can be detected for each frame so that search can be exhaustive.

Camera Feed Facial Recognition/Object Detection

Built an application to display camera feed and perform facial recognition and object detection **in real time** (24+ fps on a Mac) in Python with core C++ function calls (dlib and TensorFlow). Extensive effort was put to multithreading/processing and load balancing so that program can run fast even on a Raspberry Pi. User can toggle different states, e.g. display, training, recognizing. Can record logged-in user list and do all sorts of things. This is a project that blends AI (image/face transformation) and machine learning models (for identity classification) with good engineering and programming and can unleash a great deal of potential.

Ten Million Celebrity Faces Search

Built a state-of-the-art Web application with MongoDB and Flask to detect, transform and index features of ten million faces and allow very fast searches with user uploaded photos. The application leverages the latest computer vision neural networks (dlib), nearest neighbors and clustering algorithms (Faiss), and can be used to further derive otherwise hard-to-get information.

Political Contribution Identity Verification

Performed extensive automated webpage scraping. Queried a 20-million-entry PostgreSQL database with SQLAlchemy and pandas. Wrote complex regular-expression-based fuzzy lookups, cleaning and extraction on name, employer and occupation information of contributors. Used machine learning models to evaluate positive identification based on distance between zip code and a list of possible zip codes.

Kaggle Competition: Rental Listing Inquiries

Built machine learning classification models (SVMs, random forests, boosted trees) with scikit-learn to predict viewer interest levels of 80k rental listings. Performed text feature engineering and model comparison.

Ceramics Database

Lead a two-person team. The database is built with Django and MySQL, equipped with modern mobile Web technologies. Customized tabulation/graphing, user registration and contribution are supported. Project helped get $100,000 research grant.

Homework Collector

Website allows students submit their homework with verification and file format control. Email submission of attachments and confirmation are also allowed. Has convenient record keeping and output.

Case Study of Myopia with Logistic Regression

Selected the optimal generalized linear model and identified influencing factors such as inheritance, gender, indoors & outdoors time spent and eyeball anatomy.

Thesis project on material property prediction with cluster expansion regression models

Focused on predicting properties from atomic configurations with linear models, cross validation and selection (cluster expansion formalism).
